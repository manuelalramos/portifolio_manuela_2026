# ✦ Sobre o projeto

Este portfolio nasceu para reunir tudo o que venho construindo entre **design, desenvolvimento web e experiências digitais**.

Mais do que apenas uma vitrine, o projeto foi pensado como um espaço vivo — um ambiente para documentar evolução, processo criativo, identidade visual e projetos que desenvolvi ao longo da minha trajetória.

A proposta visual mistura uma estética mais **editorial, tecnológica e interativa**, buscando refletir tanto meu lado designer quanto meu crescimento na área de desenvolvimento front-end.

---

# ✦ Conceito

O portfolio foi desenvolvido com foco em:

- experiência visual;
- identidade autoral;
- organização modular;
- navegação fluida;
- estética clean com influência tech/editorial.

Toda a interface foi construída pensando em transformar projetos em experiências mais imersivas e visuais.

# ✦ Organização visual

A estrutura foi separada por áreas para facilitar escalabilidade e manutenção do projeto.

### Separação dos estilos

- `base.css` → variáveis globais e estilos compartilhados;
- `hero.css` → hero section;
- `about.css` → seção sobre mim;
- `menu.css` → navegação;
- `files.css` → categorias/projetos;
- `contact.css` → contato;
- `footer.css` → rodapé;
- páginas específicas possuem CSS próprio.
